dom객체와 jQuery객체 변환

jQuery를 dom으로 변환: jQuery(객체).get(N) / jQuery객체[N]
dom객체를 jQuery객체로 변환: $(dom객체)

jq = $('h1');
jq.css('color', 'blue');


- jQuery객체를 dom객체로 변환
dom = jq.get(2); //jq[2]
dom.style.color = 'red';

- dom객체를 jQuery객체로 변환
$(dom).css('color', 'green');

- 전체 h1중 4번째 선택: jQuery
jq4 = $('h1:eq(4)');
jq4.css('color', 'red');
